# 🔐 Password Strength Checker

This is a Python project that checks the strength of a password.

## 🚀 Features
- Minimum 8 characters check
- Uppercase & lowercase detection
- Numbers check
- Special characters check
- Suggestions to improve password

## 🛠️ Technologies Used
- Python
- re module

## ▶️ How to Run
python project1.py
